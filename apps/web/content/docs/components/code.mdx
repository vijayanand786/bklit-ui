# Code

An inline code component for displaying code snippets within text.

## Preview

<ComponentPreview>
  <p className="text-fd-foreground">
    Install with <Code className="rounded bg-fd-muted px-1.5 py-0.5 font-mono text-sm text-fd-primary">pnpm add @repo/ui</Code>
  </p>
</ComponentPreview>

## Installation

```bash
import { Code } from '@repo/ui/code';
```

## Usage

```tsx
import { Code } from '@repo/ui/code';

export default function Example() {
  return (
    <p>
      Run <Code className="code">npm install</Code> to install dependencies.
    </p>
  );
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | - | The code content to display |
| `className` | `string` | - | Additional CSS classes |
