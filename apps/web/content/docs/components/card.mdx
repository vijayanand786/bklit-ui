# Card

A card component for displaying linked content with a title and description.

## Preview

<ComponentPreview>
  <Card
    title="Example Card"
    href="https://example.com"
    className="block rounded-xl border border-fd-border bg-fd-card p-6 transition-colors hover:bg-fd-accent max-w-sm"
  >
    This is an example card with some descriptive text.
  </Card>
</ComponentPreview>

## Installation

```bash
import { Card } from '@repo/ui/card';
```

## Usage

```tsx
import { Card } from '@repo/ui/card';

export default function Example() {
  return (
    <Card
      title="My Card"
      href="/destination"
      className="card"
    >
      Card content goes here
    </Card>
  );
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `title` | `string` | - | The title displayed in the card header |
| `children` | `ReactNode` | - | The content/description of the card |
| `href` | `string` | - | The URL the card links to |
| `className` | `string` | - | Additional CSS classes |
