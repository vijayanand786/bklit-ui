# Button

A versatile button component for triggering actions.

## Preview

<ComponentPreview>
  <Button appName="docs" className="rounded-lg bg-fd-primary px-4 py-2 text-fd-primary-foreground hover:bg-fd-primary/90 transition-colors">
    Click me
  </Button>
</ComponentPreview>

## Installation

```bash
import { Button } from '@repo/ui/button';
```

## Usage

```tsx
import { Button } from '@repo/ui/button';

export default function Example() {
  return (
    <Button appName="my-app" className="btn-primary">
      Click me
    </Button>
  );
}
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `children` | `ReactNode` | - | The content of the button |
| `className` | `string` | - | Additional CSS classes |
| `appName` | `string` | - | The name of the app (used in alert) |
