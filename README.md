## Getting Started

```bash
pnpm install

cd apps/web && pnpm dev
```

## Project Structure

```
bklit-ui/
├── apps/
│   └── web/          # Documentation site (Next.js + Fumadocs)
└── packages/
    └── ui/           # Component library
```

## Tech Stack

- **Framework**: Next.js 15
- **Components**: React 19
- **Styling**: Tailwind CSS 4
- **Charts**: Visx
- **Animation**: Motion
- **Docs**: Fumadocs
- **Monorepo**: Turborepo + pnpm
- **Linter**: Biome/Ultracite
